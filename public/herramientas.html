HERRAMIENTAS DISCOGS

Utilidades para gestionar tu colección. Encontrá las compras que no están agregadas y generá el CSV de importación.

🔒 Tu token nunca sale de tu navegador — procesamiento 100% local
1

INGRESÁ TU TOKEN DE DISCOGS

Conseguilo en discogs.com → Settings → Developers → Generate new token. El token se usa solo en tu navegador para llamar a la API de Discogs — nunca se envía a ningún servidor externo.

Tu token de Discogs para autenticar las llamadas a la API

2

ANALIZÁ TU CUENTA

El proceso lee tu historial de compras y tu colección completa, los compara y lista los discos que compraste pero nunca agregaste a la colección. Puede tardar unos minutos si tenés muchas compras — Discogs limita las requests por segundo.

Inicia el análisis de compras vs colección en Discogs

3

RESULTADOS Y DESCARGA

Una vez completado el análisis, vas a ver la lista de discos faltantes y podrás descargar el CSV para importar directamente en Discogs.

Cómo importar el CSV: discogs.com → tu perfil → Collection → Import/Export → Upload CSV → seleccioná el archivo descargado.
